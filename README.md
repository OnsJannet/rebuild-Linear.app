## Getting Started

To run this project locally, follow these simple steps:

1. Install dependencies by running:
    ```bash
    yarn install
    ```
    or
    ```bash
    npm install
    ```

2. Start the development server by running:
    ```bash
    yarn dev
    ```
    or
    ```bash
    npm run dev
    ```

3. Access the project by clicking the link provided in the terminal.
